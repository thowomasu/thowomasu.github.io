How do I use my free subdomain?

During checkout, you will have the option to select your desired subdomain. If nothing is provided then a default subdomain will be generated for your server using the format username.server_id.frogmc.co. You can find your active subdomain in your Multicraft panel at panel.frogminecrafthosting.com on your server management page. Simply change the text, save the page, and your subdomain will be updated. Note: if a subdomain is rejected it is likely because it already exists in our system. Please contact support if you think this is in error.

Do I get a MySQL database?

Yes, SQL databases are included with all of our packages. Your Multicraft control panel will have the information required for you to create and manage your databases.

Do I get a control panel with my server?

Yes, our custom version of the Multicraft control panel is included with all game server hosting packages. This web based panel gives you complete control over your entire server with an easy to use interface. You can access the console, servers files, databases, chat and more right from here. Please contact us if you would like to try a demo of the panel before making a purchase.

What uptime do you provide?

Our hardware has over 99.9% uptime and rarely becomes unavailable. The most common cause of a crashed Minecraft server is an issue with a plugin or mod which we can help diagnose and repair.

What is my server IP?

To find your servers IP you can log into the members area and open the control panel. On the main page for the server, you will see your IP address in the format of IP:PORT. For example: 199.192.122.144:25712. Enter this address into your game client and you will be able to connect. Please make sure you include the IP and PORT when connecting.

Do I need to use port forwarding for my server?

Not at all! We make it very easy to get started on your server with very little technical knowledge needed. We will send you the IP and Port combination that you will enter into the launcher which will work without any additional configuration.

What is the difference between Minecraft Java and Bedrock?

Minecraft currently has two different versions of the game. The original Minecraft was written in Java and is now referred to as Minecraft: Java Edition which is mostly for PC. This is the most common version found on our servers and has the best support for mods, plugins and customization. Minecraft was formerly known as Minecraft: Bedrock Edition and was built on the language C and is used for The Windows 10 edition, Xbox, mobile and more.
